Picsta

This is a full-stack image storing application. This app stores a direct image link a user provides and appends the link with a img tag to the client view when the user requests the contents. Future implementation may include Amazon Web Services connectivity for direct image upload. This application was built with JavaScript, jQuery, Ruby on Rails, and Handlebars.
